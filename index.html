Micro Estação Meteorológica

Nosso projeto consiste em criar uma micro estação meteorológica com a participação de todos dividindo em três grupos.

  • Montagem
  • Banco de dados.
  • Página Web.

Página web

O grupo da página web é composto por Luana, Vinicius, Elder e Otávio

Parte do projeto consiste no desenvolvimento da página web, utilizando linguagens de programação específicas para construir a interface. Aplicamos conhecimentos adquiridos nas aulas de Desenvolvimento e Aplicações de Sistemas, além de realizar pesquisas na internet para aprender técnicas mais avançadas ainda não abordadas em sala. Atualmente, estamos desenvolvendo um protótipo que envolve não apenas HTML e CSS, mas também JavaScript, cuja utilização foi aprendida por meio dessas pesquisas.
